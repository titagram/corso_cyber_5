4. Persistenza

Deploy di massa tramite GPO
L'Attacco

Utilizzo dei privilegi di Domain Admin per creare una Group Policy Object (GPO).

La GPO installava un pacchetto MSI (java-update.msi) su tutti gli endpoint, che conteneva lo spyware.

Rilevamento (Sysmon)

Event 11: Creazione file C:\Windows\Temp\java-update.msi

Event 1: Esecuzione comando:
msiexec /i java-update.msi /quiet
