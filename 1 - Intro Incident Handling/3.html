Colonial Pipeline

Leaked Credentials

Un attacco ransomware originato da una password di un dipendente trovata nel dark web.

Causa: Accesso tramite un account VPN inattivo senza Multi-Factor Authentication (MFA).

Mirai Botnet (2016)

Default / Weak Credentials

Botnet che ha infettato dispositivi IoT usando credenziali di fabbrica (admin/admin).

Impatto: Enormi disservizi DDoS contro aziende come Dyn e OVH.

LogicMonitor (2023)

Clienti compromessi a causa di password di default deboli fornite dal vendor stesso.

Lezione: Mai lasciare credenziali di default e forzare l'hardening delle password immediatamente.
