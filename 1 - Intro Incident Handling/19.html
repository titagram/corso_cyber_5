Indicatori di Compromissione (IOC)

Le tracce lasciate dall'attaccante
Cosa sono?

Artefatti digitali che indicano una violazione:

  • Indirizzi IP malevoli.
  • Hash di file (es. malware).
  • Nomi di file o chiavi di registro.
Standardizzazione

Gli IOC possono essere condivisi usando formati standard come STIX (JSON) o regole YARA.

OpenIOC, STIX, TAXII

Attenzione!

Quando si cercano IOC sui sistemi, fare attenzione a non lasciare credenziali di amministrazione in cache (evitare PsExec con credenziali esplicite, preferire WinRM o esecuzione senza cache).
