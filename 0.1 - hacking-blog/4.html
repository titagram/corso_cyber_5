Azione Pratica

Tool:

OWASP ZAP -> Fuzzer

Payload 1 (User):

`kwheel`, `bjoel`

Payload 2 (Pass):

`rockyou.txt` (Wordlist)

Risultato:

HTTP 302 Found
Analisi Tecnica

Il Concetto: Il Fuzzing qui viene usato come un attacco a dizionario mirato.

HTTP 302: Non cerchiamo un messaggio "Login OK". Cerchiamo un codice di stato 302 (Redirect). Il protocollo HTTP usa il 302 per reindirizzare l'utente alla dashboard dopo un login corretto. Se riceviamo un 200 OK, solitamente significa che siamo ancora sulla pagina di login (tentativo fallito).
