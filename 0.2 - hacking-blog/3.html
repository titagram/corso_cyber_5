Enumerazione: Ridurre il campo di battaglia

Azione Pratica

Comando:

wpscan --url http://blog.thm -e u

Risultato:

Trovati utenti `kwheel`, `bjoel`.
Analisi Tecnica

Come funziona: WPScan non "indovina". Interroga le REST API (`/wp-json/wp/v2/users`) che, di default, espongono la lista utenti pubblica in formato JSON. Alternativamente, analizza i redirect degli archivi autore (`/?author=1`).

Perché è critico: Conoscere gli username validi trasforma un attacco Brute Force esponenziale (indovina User + Password) in uno lineare (indovina solo la Password).
