Post-Exploitation: Upgrade della Shell

Azione Pratica

Comando:

python -c 'import pty; pty.spawn("/bin/bash")'

Risultato:

Prompt colorato, supporto a `su`, `nano`, `ctrl+c`.
Analisi Tecnica

Dumb Shell vs PTY: La reverse shell base collega solo *stdin* e *stdout*. Non è un terminale vero (TTY). Comandi che richiedono interazione (come `sudo` o editor di testo) si bloccano perché non sanno gestire l'input utente.

Python Magic: Usiamo Python per invocare una Pseudo-Terminal (PTY), che simula un dispositivo terminale completo, permettendo alla shell remota di comportarsi come se fossimo seduti davanti al server.
