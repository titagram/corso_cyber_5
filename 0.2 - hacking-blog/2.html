Ricognizione Attiva: Mappare la Superficie di Attacco

Azione Pratica

Comando:

nmap -sV -sC -A 10.10.x.x

Output chiave:

Porta 80 (HTTP/WordPress)
Porta 22 (SSH)
Porta 139/445 (SMB)
Analisi Tecnica

Flag `-sV` (Service Version): Interroga le porte aperte analizzando i banner di risposta per determinare il software esatto (es. Apache 2.4.29).

Flag `-sC` (Default Scripts): Lancia script Lua di base per trovare vulnerabilità comuni.

Analisi: La presenza di SMB (condivisione file Windows/Samba) su un server Web Linux è un'anomalia. Spesso indica una configurazione errata o un vecchio backup dimenticato.
