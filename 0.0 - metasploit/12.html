Impostare i parametri necessari per l'exploit
üéØ Configurazione Target (RHOST/RHOSTS)

RHOSTS: IP address del target remoto


Impostazione temporanea:

msf6 exploit(...) > set RHOSTS 10.10.10.40 RHOSTS => 10.10.10.40

Impostazione permanente (setg):

msf6 exploit(...) > setg RHOSTS 10.10.10.40 RHOSTS => 10.10.10.40

Il comando setg rende l'impostazione permanente fino al riavvio di msfconsole.

üì° Configurazione Attacker (LHOST/LPORT)

LHOST: IP address della macchina attacker (tua)

LPORT: Porta in ascolto (default 4444)


Trovare il proprio IP:

msf6 exploit(...) > ifconfig tun0: inet 10.10.14.15 netmask 255.255.254.0

Impostare LHOST:

msf6 exploit(...) > setg LHOST 10.10.14.15 LHOST => 10.10.14.15
üìù Parametri Comuni

Target (Exploit):

  • RHOSTS - IP target (singolo o range)
  • RPORT - Porta del servizio vulnerabile
  • SMBUser - Username (se necessario)
  • SMBPass - Password (se necessario)

Attacker (Payload):

  • LHOST - IP dell'attacker
  • LPORT - Porta in ascolto
  • EXITFUNC - Tecnica di uscita
