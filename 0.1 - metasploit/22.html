Scanning e importazione automatica dei risultati
üöÄ db_nmap - Scanning Integrato

Il comando db_nmap esegue Nmap direttamente da msfconsole e salva automaticamente i risultati nel database.


Esempio completo:

msf6 > db_nmap -sV -sC -p- 10.10.10.8 [*] Nmap: Starting Nmap 7.91 [*] Nmap: Nmap scan report for 10.10.10.8 [*] Nmap: Host is up (0.016s latency). [*] Nmap: Not shown: 65534 filtered ports [*] Nmap: PORT STATE SERVICE VERSION [*] Nmap: 80/tcp open http HttpFileServer httpd 2.3 [*] Nmap: Service Info: OS: Windows; CPE: cpe:/o:microsoft:windows [*] Nmap: Nmap done: 1 IP address (1 host up) scanned in 45.23 seconds

‚úÖ I risultati sono immediatamente disponibili con hosts e services!

üìä Visualizzare Hosts
msf6 > hosts Hosts ===== address mac name os_name os_flavor os_sp ------- --- ---- ------- --------- ----- 10.10.10.8 Unknown 10.10.10.40 Windows 7 SP1

Opzioni utili:

msf6 > hosts -h -a,--add # Aggiungi host -d,--delete # Elimina host -c col1,col2 # Mostra solo colonne -u,--up # Solo hosts up -S,--search # Filtro ricerca -R,--rhosts # Imposta RHOSTS
üîå Visualizzare Services
msf6 > services Services ======== host port proto name state ---- ---- ----- ---- ----- 10.10.10.8 80 tcp http open 10.10.10.40 445 tcp microsoft-ds open 10.10.10.40 135 tcp msrpc open

Filtrare per porta:

msf6 > services -p 445

Filtrare per servizio:

msf6 > services -s http,https
üì• Importare Scan Esterni

Puoi importare scan effettuati esternamente in formato XML:


1. Eseguire scan Nmap con output XML:

$ nmap -sV -sC -oX scan_results.xml 10.10.10.0/24

2. Importare in msfconsole:

msf6 > db_import scan_results.xml [*] Importing 'Nmap XML' data [*] Import: Parsing with 'Nokogiri v1.11.1' [*] Importing host 10.10.10.8 [*] Importing host 10.10.10.40 [*] Successfully imported /root/scan_results.xml

Scanner supportati:

  • Nmap XML
  • Nessus (.nessus)
  • Nexpose XML
  • OpenVAS XML
  • Qualys XML
üéØ Usare i Dati per RHOSTS

Puoi impostare automaticamente RHOSTS dai risultati del database:

msf6 > services -p 445 -R RHOSTS => 10.10.10.40 10.10.10.50 msf6 exploit(windows/smb/ms17_010_eternalblue) > show options Name Current Setting Required ---- --------------- -------- RHOSTS 10.10.10.40 10.10.10.50 yes
