Suricata - Comandi Pratici

Input Offline e Live
Input Offline (PCAP)

Per processare file PCAP:

suricata -r /path/to/file.pcap

Opzioni aggiuntive:

  • -k none - Bypassa checksum
  • -l . - Log nella directory corrente
Input Live (LibPCAP)

Per catturare traffico live da un'interfaccia:

sudo suricata --pcap=ens160 -vv

Suricata creerà vari log (principalmente eve.json, fast.log, e stats.log) nella directory di default /var/log/suricata.

Input Live (AF_PACKET)

Per utilizzare AF_PACKET (migliore performance):

sudo suricata -i ens160 # oppure sudo suricata --af-packet=ens160

L'opzione -i aiuta Suricata a scegliere la migliore opzione di input. Nel caso di Linux, la migliore opzione è AF_PACKET.

Input Inline (NFQ)

Per modalità inline IPS con NFQ:

# Prima configura IPTables sudo iptables -I FORWARD -j NFQUEUE # Poi avvia Suricata sudo suricata -q 0

NFQ necessita di regole drop per Suricata per bloccare efficacemente i pacchetti.
