Architettura dello Stack

Interazione tra componenti
Logstash: Il Collettore

Input: Riceve dati da file, syslog, TCP.

Filter: Trasforma, arricchisce e normalizza i log.

Output: Invia i dati a Elasticsearch.

Beats: I Data Shippers

Agenti leggeri installati sugli endpoint per inviare dati (es. Winlogbeat per log Windows, Filebeat per file).

Beats -> (Logstash) -> Elasticsearch -> Kibana

Elasticsearch & Kibana

Elasticsearch indicizza i dati JSON rendendoli ricercabili via API RESTful. Kibana interroga Elasticsearch per creare dashboard, grafici e permettere l'analisi visuale.
