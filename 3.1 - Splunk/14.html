Identificare i Dati Disponibili

Approccio 2: Utilizzare l'Interfaccia Utente
Data Sources

Per identificare le fonti dati:

  1. Naviga al menu Settings (angolo superiore destro)
  2. Seleziona Data inputs
  3. Vedrai una lista di vari metodi di input dati:
    • Files & directories
    • HTTP event collector
    • Forwarders
    • Scripts
    • E molti altri...
Data (Events)

Per identificare gli eventi:

  1. Naviga all'app Search & Reporting
  2. Esplora gli eventi in Fast mode per scansione rapida
  3. Usa Verbose mode per dettagli approfonditi di ogni evento, inclusi dati raw e tutti i campi estratti
  4. Nella barra di ricerca, puoi inserire * per vedere tutti i dati indicizzati (usa il time picker per selezionare un intervallo più piccolo)
Fields

Per identificare i campi:

  1. Clicca su qualsiasi evento nei risultati per espanderlo
  2. Sul lato sinistro vedrai due categorie:
    • Selected Fields: Campi sempre mostrati (host, source, sourcetype)
    • Interesting Fields: Campi che appaiono in almeno il 20% degli eventi
  3. Cliccando All fields, vedrai tutti i campi presenti negli eventi
Data Models

I Data Models forniscono una vista organizzata e gerarchica dei dati.

Per accedere:

  1. Settings → Data Models (sezione Knowledge)
  2. Visualizza la struttura gerarchica divisa in oggetti
  3. Ogni oggetto rappresenta una parte specifica dei dati e contiene campi rilevanti
