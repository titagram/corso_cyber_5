Componenti Chiave di Splunk

Elementi Fondamentali della Piattaforma
Splunk Web Interface

Interfaccia grafica attraverso cui gli utenti interagiscono con Splunk.

Funzionalità:

  • Ricerca dati
  • Creazione di alert
  • Dashboard personalizzate
  • Report e visualizzazioni
Search Processing Language (SPL)

Il linguaggio di query per Splunk, contenente oltre cento comandi, funzioni, argomenti e clausole.

Utilizzo:

  • Ricerca dati indicizzati
  • Filtraggio eventi
  • Trasformazione dati
  • Visualizzazione risultati

Backbone dell'analisi dati in Splunk.

Apps e Add-ons

Apps: Forniscono funzionalità specifiche all'interno di Splunk. Consentono la coesistenza di più workspace su una singola istanza Splunk.

Technology Add-ons: Strato di astrazione per metodi di raccolta dati. Includono estrazioni di campi rilevanti, file di configurazione (props/transforms) e script/binari di supporto.

Splunkbase: Repository di app pronte all'uso e soluzioni pre-configurate.

Knowledge Objects

Oggetti che arricchiscono i dati in Splunk, facilitando ricerca e analisi:

  • Fields: Campi estratti o definiti manualmente
  • Tags: Etichette per categorizzare eventi
  • Event Types: Classificazioni di eventi
  • Lookups: Tabelle di riferimento
  • Macros: Query riutilizzabili
  • Data Models: Modelli strutturati
  • Alerts: Notifiche automatiche
