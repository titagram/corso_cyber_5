Ottimizzazione delle Query
Identificare Source Types

Iniziamo elencando tutti i nostri sourcetypes per approcciare questo come un ambiente sconosciuto da zero.

Query:

index="main" | stats count by sourcetype

Questo elencherà tutti i sourcetypes disponibili nel tuo ambiente Splunk.

Performance delle Query

Query generica (lenta):

index="main" *uniwaldo.local*

Questa query restituisce risultati molto più lentamente, anche se il numero di risultati è esattamente lo stesso!

Query Mirata (veloce)

Query mirata al campo:

index="main" ComputerName="*uniwaldo.local"

Questa query restituisce risultati molto più rapidamente!

Lezione Importante

Le ricerche mirate nel tuo SIEM:

  • Si eseguono e restituiscono risultati molto più velocemente
  • Riducono il consumo di risorse
  • Permettono ai colleghi di usare il SIEM con meno interruzioni
  • Sono cruciali quando si convertono query in alert

Regola d'oro: Se puoi mirare la ricerca a utenti, reti, macchine specifiche, ecc., sarà sempre a tuo vantaggio farlo, poiché riduce anche molti dati irrilevanti, permettendoti di concentrarti su ciò che conta davvero.
