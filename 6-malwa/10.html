Windows API

Il linguaggio del malware
Cosa sono le API?

Le Windows API (o WinAPI) sono le funzioni che Microsoft mette a disposizione dei programmatori per interagire con l'OS.

Il malware DEVE usarle. Non può magicamente rubare un file o aprire una connessione di rete senza invocare una API.

Esempi Comuni
  • CreateFile / ReadFile (Rubare dati)
  • InternetOpenUrl (Contattare C2)
  • RegSetValueEx (Persistenza nel registro)
