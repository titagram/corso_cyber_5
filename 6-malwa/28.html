Automazione: Noriben

ProcMon sotto steroidi
Il Problema

ProcMon cattura anche il rumore di fondo di Windows (servizi, update, indicizzazione). Trovare il malware è difficile.

La Soluzione Noriben

Uno script Python che:

  1. Lancia ProcMon.
  2. Applica filtri per escludere il rumore noto di Windows.
  3. Esegue il malware.
  4. Genera un report pulito (TXT/HTML) solo con le attività sospette.
