MAC Address e IP Address
MAC Address

Ogni interfaccia logica o fisica collegata a un host ha un Media Access Control (MAC) address.

Formato: 48-bit, sei ottetti rappresentati in formato esadecimale

Utilizzo: Comunicazioni Layer 2 (data-link/link-layer) tra host all'interno di un dominio di broadcast

Funzionamento MAC

Il traffico Layer 2 funziona attraverso comunicazione host-to-host all'interno di un dominio di broadcast.

Se il traffico Layer 2 deve attraversare un'interfaccia Layer 3, il PDU viene inviato all'interfaccia egress Layer 3 e viene instradato alla rete corretta.

Il router sostituisce l'incapsulamento Layer 2 con nuove informazioni che indicano il prossimo indirizzo fisico nel percorso.

Esempio MAC Address

Un MAC address appare come: 00:0c:29:97:52:65
