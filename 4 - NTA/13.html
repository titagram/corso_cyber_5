FTP - File Transfer Protocol

Protocollo di Trasferimento File
Caratteristiche FTP

Protocollo del livello Applicativo che abilita il trasferimento rapido di dati tra dispositivi informatici.

Nota: FTP è stabilito come protocollo non sicuro. La maggior parte degli utenti è passata a strumenti come SFTP per trasferire file attraverso canali sicuri.

La maggior parte dei browser web moderni ha eliminato il supporto per FTP a partire dal 2020.

Porte e Modalità

Porte utilizzate:

  • Porta 20: Trasferimento dati
  • Porta 21: Comandi di controllo

Modalità:

  • Attiva: Default, server ascolta comando PORT dal client
  • Passiva: Client invia PASV, server risponde con IP e porta
Comandi FTP Comuni
Comando Descrizione
USER Specifica l'utente per il login
PASS Invia la password per l'utente
LIST Mostra lista file nella directory corrente
RETR Recupera un file dal server FTP
QUIT Termina la sessione
