Funzionalità e Persistenza

Cosa fanno una volta dentro?
RAT (Remote Access Trojan)

Modulare ed espandibile. Capacità osservate:

  • Screen Capture
  • Mimikatz (furto password)
  • Interactive Shell (CMD)

Persistenza

File EXE droppato su disco. Spesso con nomi comuni per nascondersi.

Lateral Movement

Si muovono nella rete usando strumenti legittimi (LotL - Living off the Land):

  • PsExec (firmato Microsoft)
  • WinRM (Windows Remote Management)
