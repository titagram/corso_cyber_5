Ciclo d'Attacco

Initial Access & Execution
1. Phishing Email

Email con link a file OneNote hostati su servizi legittimi (es. Mega.io, Transfer.sh).

2. OneNote Malicious

Il file si finge una fattura. Contiene un pulsante "HIDDEN" che attiva un file Batch nascosto.

3. Payload

Il Batch scarica ed esegue script PowerShell (Stage 0). Questo porta a un RAT modulare.
