I 3 Livelli di Intelligence

Strategic, Operational, Tactical
Strategic

Per chi: C-Level, Executive.
Cosa: Trend a lungo termine, rischi geopolitici, motivazioni (Why/Who).
Esempio: "Il gruppo APT28 targettizza le aziende energetiche."

Operational

Per chi: SOC Manager, Security Team.
Cosa: Campagne attive e TTPs (How/Where).
Esempio: "Campagna ransomware REvil usa RDP per muoversi lateralmente."

Tactical

Per chi: Analisti SOC, Firewall.
Cosa: IoC tecnici immediati.
Esempio: IP malevole 192.168.1.5, Hash malware MD5.
